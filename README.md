
# To do stuff

### Clear and basic api based to do list.

## [LIVE DEMO](https://todostuff.surge.sh/)

### Build with:
- React
- Typescript
- Redux

bootstrapped with CRA

# Installation 

### Clone

- Clone this repo to your local machine using `https://github.com/patryk-urbanski/todo_some_stuff.git`

### Setup

```shell
$ yarb install
$ yarn start
```
